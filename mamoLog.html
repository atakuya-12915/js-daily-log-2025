Todoシンプル版（MVP）

Spring Boot + Thymeleaf で作ったシンプルなタスク管理アプリ

概要

未完了タスクと完了タスクを分けて管理できるシンプルなTodoアプリです。 Home画面風のカードデザインで、CRUD操作と完了切替が可能です。初学者でも理解しやすい設計を意識しました。

主な機能

  • タスク一覧表示（未完了・完了の分離、件数表示）
  • タスク新規作成（タイトル・メモ・担当者・期限日・時刻・完了フラグ）
  • タスク編集・削除
  • 完了切替（チェックボックス操作で即座にUI更新）
  • 完了タスクは折りたたみ表示可能
  • Home画面風のカードデザイン統一

スクリーンショット

使用技術

Spring Boot Thymeleaf JavaScript (DOM操作, fetch API) CSS3 / Flexbox Lombok H2 / MySQL / PostgreSQL

学習ポイント / 工夫した点

  • チェックボックス操作で非同期にDOMを移動させ、即座にUIを更新
  • 完了タスクの折りたたみ表示と件数更新の連動
  • CSS共通化でフォーム・カードのデザインを統一
  • 初学者でも理解しやすい構成でMVCモデルを実装
  • エラー対応やイベント再バインドを通じて実務に近い感覚を学習

想定質問と回答例

  • Q: JSで完了切替を実装した理由は？
    A: 非同期でUIを即座に更新し、ユーザー体験を向上させるため。サーバー応答後にDOMを移動させています。
  • Q: Thymeleafを選んだ理由は？
    A: サーバーサイドで変数を埋め込み、簡潔にMVCパターンで画面を生成できるためです。
  • Q: CSSを共通化した意図は？
    A: Home画面とフォームのデザインを統一し、保守性を高めるためです。
